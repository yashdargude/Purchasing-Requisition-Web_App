## Frontend Installation
 ## technologies used 
   - React 
   - Vite as bundler
   - typescript
   - Shadcn for component library
   - tailwind as css framework
   - axios as API requests
   - npm 
   - node


### Run Locally



Go to the project directory

```bash
  cd UI
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
